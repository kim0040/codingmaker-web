## 🐛 오류 체크리스트

### 콘솔 에러 확인
1. 브라우저 F12 → Console 탭
2. 빨간 에러 메시지 확인
3. Network 탭에서 404 또는 500 에러 확인

### 일반적인 문제

| 문제 | 원인 | 해결 방법 |
|------|------|----------|
| 페이지 404 | 라우트 없음 | 파일 생성 확인 |
| API 에러 | 백엔드 미실행 | 백엔드 서버 시작 |
| 스타일 깨짐 | CSS 로딩 실패 | 서버 재시작 |
| Textarea 에러 | 컴포넌트 미인식 | 서버 재시작 |
| 애니메이션 안 보임 | CSS 미적용 | 캐시 삭제 후 새로고침 |

### 해결 방법
```bash
# 캐시 삭제 및 재시작
rm -rf .next
npm run dev
```
