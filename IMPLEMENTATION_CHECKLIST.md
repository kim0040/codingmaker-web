## 📋 명세서 요구사항 매핑

### 사용자 권한 체계
| Tier | 역할 | 구현 상태 |
|------|------|----------|
| 1 | 최고 관리자 (원장) | ⚠️ UI만 구현, 권한 체크 없음 |
| 2 | 관리자 (강사) | ⚠️ UI만 구현, 권한 체크 없음 |
| 3-A | 정회원 (수강생) | ✅ 기본 UI 완료 |
| 3-B | 학부모 | ✅ 기본 UI 완료 |
| 3-C | 명예회원 | ❌ 미구현 |
| 4 | 비정규회원 | ❌ 미구현 |
| 5 | 게스트 | ✅ 홈페이지만 접근 가능 |

### 핵심 기능
| 기능 | 명세서 요구사항 | 현재 상태 |
|------|----------------|----------|
| 동적 커리큘럼 | 관리자가 추가/수정 가능 | ⚠️ 하드코딩됨 |
| 출석 시스템 | 음성 안내, 학부모 알림 | ⚠️ UI만, 백엔드 필요 |
| 암호화 데이터 | AES-256 복호화 후 표시 | ❌ 준비 안됨 |
| Footer | DB에서 동적 로드 | ❌ 하드코딩됨 |
| 데이터 분석 | Recharts 그래프 | ❌ 미구현 |
